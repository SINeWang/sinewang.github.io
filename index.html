  3. 守住自己的心

    Mon 14 April 2014
    By Wang YanJiong

    上学的时候，课本教育我们要唯物，也就是客观的去理解、评论一件事一个物，更似乎是有些带着贬低唯心的去追崇唯物。可是客观到底有多客观，到底是相对客观还是绝对客观，这首先取决于我们每个人的认知（客观），再取决于认知之上的心（主观）。

    人生是什么？有意义的人生又是什么？有太多的人，太多的大家，太多的哲人去探究过这个问题，可是始终没有一个普适的、完美的答案。也许因为很多时候这个答案都取决于我们的认知，以及我们的心，而每个人的认知和心又是如此的不同。

    最近时常想起四个字“心平气和”，是的，心要平了，气才能和。可是，心为什么会不平呢？

    那"心"是什么，在这片短文里，我想简单的把它解释成“欲望”。

    所以心不平，在这里就可以解释成“期待”和“现实”的不对等。

    那欲望又是什么，当然欲望可以分为物质欲望和精神欲望，在这里，我对欲望的理解是“愉悦”，而愉悦是来自于人的主观感受，来自于自己的心。

  4. 在MacOS X 中安装qRFCView

    Sun 14 October 2012
    By Wang YanJiong

    环境

    • Mac OS X 10.8.2
    • Macports 2.1.2
    • qRFCView 0.62

    条件

    • 必须先安装Qt环境

    步骤

    1. download qRFCView
    2. $sudo port install qt4-mac（安装Qt4）
    3. $cd <path/to/qrfcview-0.62>
    4. $qmake rfcview.pro
    5. $make
    6. 编译后在<path/to/qrfcview-0.62/bin>下生成 qRFCView.app
    7. 运行$open qRFCView.app
