HHKB 在 macOS 上的使用速记

键帽

  • 空格键的键帽，最好拆卸来之后反装，否则容易伤到大拇指

光标移动组合键

其实下面这些快捷键，都是macOS自带的快捷键

  • 上一行 CTRL + p
  • 下一行 CTRL + n
  • 行首 CTRL + a
  • 行末 CTRL + e
  • 删除后一个字符 CTRL + d
  • 删除前一个字符 CTRL + h
  • 删除光标之后的所有内容 CTRL + k
  • 交换光标左右两个字符 CTRL + t
  • 移到下一个字符 CTRL + f
  • 移到下一个字符 CTRL + b
