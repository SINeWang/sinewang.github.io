  1. 在MacOS X 中安装qRFCView

    Sun 14 October 2012
    By Wang YanJiong

    环境

    • Mac OS X 10.8.2
    • Macports 2.1.2
    • qRFCView 0.62

    条件

    • 必须先安装Qt环境

    步骤

    1. download qRFCView
    2. $sudo port install qt4-mac（安装Qt4）
    3. $cd <path/to/qrfcview-0.62>
    4. $qmake rfcview.pro
    5. $make
    6. 编译后在<path/to/qrfcview-0.62/bin>下生成 qRFCView.app
    7. 运行$open qRFCView.app
